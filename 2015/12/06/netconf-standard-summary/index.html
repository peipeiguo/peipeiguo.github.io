NETCONF标准总结

发表于 2015-12-06 | 分类于 NETCONF | 阅读次数

简介

目的

本文总结了IETF NETCONF工作组关于NETCONF协议的相关标准和草案。

范围

IETF NETCONF工作组NETCONF协议相关的标准，只关注重要的标准和草案。

标准分类

协议机制

描述 RFCs
NETCONF协议1.0版本 RFC 4741, RFC 4742, RFC 5539
NETCONF协议1.1版本 RFC 6241, RFC 6242, RFC 7589
RESTCONF协议 RFC 8040
NETCONF事件通知 RFC 5277
NETCONF访问控制(NACM) RFC 6536

协议扩展

描述 RFCs
NETCONF部分锁rpc操作 RFC 5717
NETCONF with-default能力集 RFC 6243

自动部署

描述 RFCs
NETCONF Call Home draft-ietf-netconf-call-home
基于NETCONF Call Home的ZTP draft-ietf-netconf-zerotouch

数据模型

描述 RFCs
NETCONF Server状态监控数据模型 RFC 6022
NETCONF Server基础通知数据模型 RFC 6470
NETCONF Server配置数据模型 draft-ietf-netconf-server-model
YANG模块库数据模型 draft-ietf-netconf-yang-library
